修改头像

string uc_avatar(integer uid)

函数参数
参数 含义
integer uid 用户 ID
返回值
值 含义
string 修改头像的 HTML 代码

本接口函数用于生成可修改用户头像的 HTML 代码，生成的 HTML 显示效果类似如下效果。

头像格式

头像尺寸 调用方法
200 x 400 {UC_URL}/avatar.php?uid={N}&size=big
120 x 240 {UC_URL}/avatar.php?uid={N}&size=middle
48 x 48 {UC_URL}/avatar.php?uid={N}&size=small
{UC_URL} 为 UCenter 的 URL。{N} 为用户 ID。

例如调用用户 ID 为 58 用户的 200 x 400 尺寸头像方法为： 
<img src="http://www.yourwebsite.com/uc_server/avatar.php?uid=58&size=big" />
